# QBO Direct URLs & Navigation Shortcuts

Save time by going directly to the right page. Replace `[COMPANY_ID]` with the customer's company ID (visible in the QBO URL bar after `/app/`).

---

## How to Find the Company ID

The company ID appears in the QBO URL: `https://qbo.intuit.com/app/[COMPANY_ID]/...`

It is a numeric string (e.g., `123456789012345`). The customer can see it in their browser's address bar when logged into QBO.

## Keyboard Shortcuts (QBO Web)

QBO has limited keyboard shortcuts, but these work:

| Shortcut | Action |
|----------|--------|
| `Ctrl + Alt + I` | Create Invoice |
| `Ctrl + Alt + X` | Create Expense |
| `Ctrl + Alt + W` | Create Check |
| `Ctrl + Alt + C` | View Customers |
| `Ctrl + Alt + V` | View Vendors |
| `Ctrl + Alt + A` | View Chart of Accounts |
| `Ctrl + Alt + H` | Help |
| `/` (forward slash) in search | Focus the global search bar |

Note: These shortcuts may vary by browser and OS. They work most consistently in Chrome on Windows.
