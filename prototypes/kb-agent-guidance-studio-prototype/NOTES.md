# KB Agent Guidance Studio Prototype Notes

This is a standalone prototype for the premium KB brief. It does not edit or depend on production app files.

## Product Job

The user is deciding whether a piece of knowledge can become reusable guidance for specialist agents. The first screen makes the current decision visible, gives one dominant outcome, and keeps evidence and audit detail available without making those details the main workspace.

## Prototype Lens

- Left pane: knowledge items that need a human decision.
- Center pane: guidance fields the user can correct before publication.
- Permission toggles: plain-language controls for what agents may do with the knowledge.
- Right pane: live preview of what agents receive plus publish blockers.
- Drawer: evidence and audit trail as secondary detail.

## What It Deliberately Avoids

- It does not copy the image-parser workflow.
- It does not use dashboards, metrics, lifecycle strips, or backend state language as the main UI.
- It does not treat the KB as the whole product. The KB is framed as governed memory for specialist agents.

## How To Open

Open `index.html` directly in a browser. No dev server is required.
