# Project Memory

## Persistent User Preference

- Build prototypes as standalone pages in a separate folder.
- Use plain files: `index.html`, `styles.css`, and `script.js`.
- Keep prototype work isolated from the main React/Express app unless explicitly requested.

## Prototype Location Convention

- Store prototypes under: `prototypes/<prototype-name>/`
- Minimum file set:
  - `prototypes/<prototype-name>/index.html`
  - `prototypes/<prototype-name>/styles.css`
  - `prototypes/<prototype-name>/script.js`

## Working Rule

- If a request is "create a prototype", default to creating a new folder in `prototypes/` with standalone HTML/CSS/JS files.
